# Meyparking — Documentación legal pública

Sitio estático con la documentación legal de la aplicación **Meyparking** de **Maquinaria Estacionamientos y Parkímetros, S.L.** (MEYPAR).

Publicado con GitHub Pages únicamente para cumplir el requisito de Google Play Console de vincular una URL pública de privacidad. La aplicación es de **uso interno MEYPAR** y se distribuye por track privado de Google Play.

## URLs públicas

- Portada: https://meyparjeffrey.github.io/meyparking-legal/
- Política de privacidad: https://meyparjeffrey.github.io/meyparking-legal/privacy-policy-es.html
- Términos de uso: https://meyparjeffrey.github.io/meyparking-legal/terms-of-service-es.html

## Contenido

Los textos en este repositorio son **idénticos** a los que la app muestra dentro de sus pantallas embebidas de Ajustes → Legal. La fuente de verdad es la app (`app/src/main/res/raw/privacy_policy_es.md` y `terms_of_service_es.md` del repositorio privado de código). Cuando se actualice la app, este sitio se actualizará en la misma release.
